# 🏪 Dormitory Shop Bot

Телеграм-бот для заказа продуктов в общежитии колледжа. Пользователи выбирают товары из каталога, добавляют в корзину и оформляют доставку прямо в комнату.

---

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone
cd PigeonTestBot
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Linux / macOS
```

### 3. Установить зависимости

```bash
pip install aiogram asyncpg python-dotenv
```

### 4. Создать `.env` файл


### 5. Запустить бота

```bash
python main.py
```

---


## Функции для пользователя

- 🏪 **Магазин** — каталог с категориями и подкатегориями, пагинация по 8 товаров
- 🔍 **Поиск** — поиск товара по части названия
- 🛒 **Корзина** — добавление, изменение количества, очистка, оформление заказа
- 📦 **Доставка** — выбор способа (в руки / у двери), сохраняется в профиле
- 👤 **Личный кабинет** — история заказов, номер комнаты, тип доставки
- 👥 **Реферальная программа** — личная ссылка, 10% бонус с заказов рефералов
- 💬 **Поддержка** — написать сообщение администратору, получить ответ в боте

---

## ⚙️ Функции администратора

Доступны по команде `/admin` (только для `ADMIN_IDS` из `.env`):

- 📊 **Статистика** — пользователи, заказы, выручка за всё время и за сегодня
- 📦 **Товары** — список всех товаров, редактирование названия/цены, снятие с продажи, удаление
- ➕ **Добавить товар** — выбор категории, название, цена, подтверждение
- 🗂 **Добавить категорию** — название и опциональный родитель
- 📨 **Сообщения** — просмотр обращений от пользователей, ответ от имени бота, очистка
- 📢 **Рассылка** — отправка сообщения всем пользователям с предпросмотром
